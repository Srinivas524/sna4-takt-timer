# SNA4 Takt Time Study Timer

A floating time study timer for the FCLM Portal with associate management
and process path selection.

## How to Install

### 1. Install Tampermonkey
Download for your browser:
- [Chrome](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd)
- [Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)

### 2. Install the Script
👉 **[Click here to install](https://raw.githubusercontent.com/YOUR_USERNAME/sna4-takt-timer/main/sna4-takt-timer.user.js)**

Click "Install" when Tampermonkey asks.

### 3. Use It
1. Go to [fclm-portal.amazon.com](https://fclm-portal.amazon.com)
2. Look for the **purple clock button** in the bottom-right corner
3. Click it to open the timer

## Keyboard Shortcuts
| Shortcut | Action |
|----------|--------|
| `Alt + T` | Open/close the timer |
| `Space` | Record time (while timing) |
| `Alt + ←/→` | Switch between associates |
| `Escape` | Close panel |

## Process Paths
- **Pick** — Singles, VNA 1, VNA 2, Noncon/Bod, Multi
- **Pack** — Singles/VNA, Multies, BOD/Noncon
- **Stow** — Standard (no sub-paths)

## Features
- ⏱ Click-based stopwatch timing
- 👥 Multiple associate management
- 📊 5 observations per process path
- 📝 Coaching notes per associate
- 📥 CSV export & clipboard copy
- 💾 Auto-saves to browser storage
- 🔄 Auto-updates when new versions are pushed
